ArcGIS Map and Image Services

Map and Image Services make map data (vector and raster formats) available on the web. ArcGIS services enable internet users to add data to web-applications, ArcGIS Desktop, and ArcGIS Online.

CartoDB SQL API Calls

The CartoDB SQL API allows you to interact with GFW data inside CartoDB using SQL statements. You can learn more about the API here.

PRO TIP: Add "&format=shp" to the end of an API call to initiate a download from the CartoDB database (for example, download Tiger Conservation Landscapes)

WMS

Web Map Services (WMS) are a OGC standard protocol for serving georeferenced map images over the web. Similar to the ArcGIS Services, WMS are generated by a map server using data from a GIS database.
